This folder contains the final report, all additional MATLAB code and all results mentioned in the report.

/App contains all functions necessary to run Interactive App v0.6

/Experiments contains all functions for experiments 1-4 and their test cases.

/Results contain all the Use Cases from the report.


Each result folder corresponds to a result mentioned in the final report and includes the following:

	soundAmp output sound (saved as a .wav)
	useCase.m (which can be loaded into the app)
	The Exported Plots (an FFT and spectrogram) (saved as a .png)
	PM_MDOF_RUN_REPORT.txt shows the relevant useCase information, including app the parameters used.
